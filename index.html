Assyrian Phonetics Tool

This tool can be used to supplement learning Assyrian. It is intended to be used for verifying word pronunciations and transliterations. There is a good chance there are bugs, if you see anything off you can let me know.

This tool can be run completely offline. Both the Syriac-to-IPA conversion and the IPA-to-phoneme audio synthesis run on your machine locally without any remote connections. The text converter currently supports all standard atwateh, zoweh, rukakheh/qushayeh, majleaneh, siyameh, and feminizing dots.

Refer to the IPA chart for pronunciations.
